# AWS EC2 Static Website Deployment

A cloud computing project focused on deploying and hosting a static website using Amazon EC2 and Apache Web Server on Ubuntu Linux.

## Overview

This project demonstrates the fundamentals of cloud-based web hosting by deploying a simple HTML website on an AWS EC2 virtual machine and making it publicly accessible through a public IP address.

The goal was to gain hands-on experience with cloud infrastructure, Linux server administration, and web server configuration.

## Features

* AWS EC2 virtual machine deployment
* Ubuntu Linux server configuration
* Apache Web Server installation and setup
* Static website hosting
* Public accessibility via IP address
* HTTP traffic configuration through Security Groups
* Remote server management using SSH

## Technologies Used

* Amazon EC2
* Ubuntu Linux
* Apache HTTP Server
* HTML5
* SSH
* AWS Security Groups

## Project Workflow

1. Created and launched an AWS EC2 instance.
2. Configured the Ubuntu Linux environment.
3. Connected to the server securely using SSH.
4. Installed and configured Apache Web Server.
5. Developed a simple HTML webpage.
6. Configured Security Groups to allow HTTP traffic.
7. Deployed the website and verified public accessibility.

## Result

Successfully hosted a static website displaying:

> Welcome to DecodeLabs

The website was accessible through the EC2 instance's public IP address, demonstrating a complete cloud-hosted web deployment.

## Skills Gained

* Cloud Computing Fundamentals

* SSH Connectivity and Security
* Web Hosting and Deployment
